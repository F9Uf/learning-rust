# Learn - RUST
ref: https://doc.rust-lang.org/stable/book/title-page.html

## Installation
```shell
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

## Command
- to initial project
    ```
    $ cargo int
    ```
    or 
    ```
    $ cargo new <project-name>
    ```

- to build
    ```
    $ cargo build
    ```

- to run
    ```
    $ cargo run
    ```
